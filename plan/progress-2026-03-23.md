# MARPLE Progress Report — 23 March 2026

## Summary

Fourth session. MicroPython port working on Raspberry Pi Pico 2, web REPL
with Bootstrap UI, PyPI publication, and comprehensive MkDocs documentation.

## What was built this session

### MicroPython port for Raspberry Pi Pico 2
- Systematic removal of CPython-only features across 18+ source files:
  - `from __future__ import annotations` removed everywhere
  - `dataclasses` replaced with plain classes (`__init__` + `__eq__`)
  - `enum.Enum` replaced with string constants for `TokenType`
  - `importlib.import_module` replaced with `__import__` + getattr traversal
  - `itertools.product` replaced with local recursive generator
  - `os.path` replaced with `_isdir()` (using `os.stat`) and `_join()` helpers
  - `os.environ` wrapped in try/except throughout
  - `str.isdigit/isalpha/isalnum` replaced with helper functions
  - `str.rjust` replaced with `_rjust()` helper
  - `np.tensordot` guarded with `hasattr` for ulab compatibility
  - f-strings with nested quotes replaced with string concatenation
- Hex-encoded UTF-8 serial protocol (MicroPython `input()` drops non-ASCII)
- `scripts/pico_eval.py` — REPL loop deployed as `main.py` on Pico
- `scripts/pico_client.py` — workstation-side serial client
- `scripts/pico_demo.py` — typing simulator for demo recordings
- `scripts/deploy.sh` — deploys all source + stdlib via mpremote
- Verified working: arithmetic, vectors, matrices, dfns, reduce, scan,
  outer product, string functions (`$::str::upper/lower/trim`), file I/O

### Web REPL
- Bootstrap 5 dark theme with session area + workspace sidebar
- Language bar with clickable APL glyphs (backtick input removed — unreliable)
