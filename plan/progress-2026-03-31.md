## Design Decisions

- **Core code never imports adapters** — only ABCs. Adapters are created at entry points.
- **MicroPython ignores annotations** — no need for `from __future__ import annotations` or `TYPE_CHECKING` guards.
- **abc stub on Pico** makes ABC imports safe on all platforms.
- **`⎕`/`⍞` input on Pico** requires PRIDE `--pico-port` mode — the serial protocol is too simple for mid-evaluation input.
