## What was built today

### Numpy backend
- Auto-detects numpy, CuPy, or ulab (CircuitPython/MicroPython)
- Falls back to pure Python when no backend available
- Pervade functions use numpy ufuncs: **73x faster** for element-wise ops on large arrays
- Reduce uses `ufunc.reduce()` for commutative operations (~6.5x faster)
- Inner product `+.Г—` uses `np.dot` for matrix multiply
