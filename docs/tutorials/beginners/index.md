# Beginner Tutorials

These tutorials assume you can program in some language but have never used APL. Work through them in order — each one builds on the previous.

1. **[Arrays — Scalars, Vectors, Matrices](arrays.md)** — APL's fundamental data structure
2. **[Scalar Functions and Scalar Extension](scalar-functions.md)** — how arithmetic (and more) works on arrays
3. **[Reduce and Scan](reduce-and-scan.md)** — your first operators: inserting a function between elements
4. **[Reshape and Iota — Building Arrays](reshape-and-iota.md)** — creating arrays of any shape
5. **[Reading APL Right to Left](right-to-left.md)** — the evaluation rule that replaces precedence
6. **[Writing Your First Dfn](first-dfn.md)** — defining your own functions

After completing these, you'll be comfortable reading and writing basic MARPLE expressions. Continue with the [intermediate tutorials](../intermediate/index.md) for the rank operator, From indexing, and more.
